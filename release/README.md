# Releases

## v1.0.0 on 2025/10/12

The first version.

Features:

- Support focus/short break/long break work mode
- An intuitive countdown progress ring that supports clicking pause and continue
- Support setting timer duration with seconds as the granularity
- Support alarm notification

## v1.1.0 on 2025/11/24

New features:

- Redesigned UI with the material design language
- Independent Timer, Statistics, Settings and About pages
- A Sidebar which can be switched between hidden and displayed
- Supports the world's most widely used languages, up to fifteen
- Supports dark and light themes that switch along with the system
- Supports windows app notification
