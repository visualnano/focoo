## 簡介

Focoo 是一款基於番茄工作法的簡單直觀的效率應用，旨在幫助您更好地專注於任務並提高工作效率。當你完成既定專注任務後你將會獲得積極的正向反饋。

讓Focoo帶你告別拖延症，逐步實現自我突破！

適用場景：
學習和複習
工作和辦公
個人自律
習慣養成
冥想練習
...

功能特點：
 介面簡潔，操作直觀
 功能專一，軟體包體積小
 靈活設定時長，精確到秒
 靈活可自訂，配置項豐富
